# Sales Analysis Project

## Overview
This project analyzes sales data of a company using MySQL and Excel.
The dataset contains 2,448 rows of sales transactions across multiple countries and product lines.

## Tools Used
- MySQL Workbench — Data storage and SQL analysis
- Microsoft Excel — Dashboard and data visualization

## Dataset
- Source: Kaggle — Sample Sales Data by kyanyoga
- Rows: 2,448
- Columns: 25

## Analysis Performed
1. Sales by Year
2. Revenue by Country
3. Revenue by Product Line
4. Top 10 Customers
5. Sales by Deal Size
6. Sales by Month

## Key Insights
- 2004 was the best sales year with 4,724,163 in revenue
- USA is the top country generating 3,627,983 in revenue
- Classic Cars is the best selling product line
- Euro Shopping Channel is the biggest customer
- Medium deals contribute 60% of total revenue
- November is the peak sales month

## Files in this Repository
- sales_analysis_queries.sql — All SQL queries used
- Sales_Analysis_Dashboard.xlsx — Excel dashboard with charts and insights
